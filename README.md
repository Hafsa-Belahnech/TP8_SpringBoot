# TP 8 : Spring Boot
> Cours : JakartaEE -> Spring
  

# Objectifs de cette application :
 - Enregistrer un étudiant dans la base de données avec MySQL
 - Afficher tous les étudiants
 - Rechercher un étudiant par identifiant, modifier ses informations, ou le supprimer 
 - Tester toutes les opérations et modifications à l'aide de Swagger UI (documentation)


# Technologies utilisées :
- JDK 17 & Maven 
- MySQL
- IntelliJ

# Dépendances requises pour ce tp :
- Spring Data JPA, Spring Web, Spring Boot
- MySQL Driver
- Swagger / OpenAPI
